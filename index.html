Desert Interaction Effects Scene

I tried to create a realistic, interactive desert scene by implementing several key effects. These include reflective sand that reacts dynamically when facing the camera, ray marching achieved by modifying the Unity URP pipeline, and footprint effects rendered using customized PBR shaders. You can explore the source code on GitHub.
