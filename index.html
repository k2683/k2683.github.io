Desert Interaction Effects Scene 沙漠交互特效场景

I tried to create a realistic, interactive desert scene by implementing several key effects. These include reflective sand that reacts dynamically when facing the camera, ray marching achieved by modifying the Unity URP pipeline, and footprint effects rendered using customized PBR shaders. You can explore the source code on GitHub. 我尝试通过实现几个关键效果来创建一个逼真、可交互的沙漠场景。这些效果包括动态反应的反射沙子，通过修改Unity URP管线实现的光线行进，以及使用自定义PBR着色器渲染的脚印效果。您可以在GitHub上查看源代码。
